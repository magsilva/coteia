1. A indentação deve utilizar tabulações (preferencialmente) ou espaços
em branco (oito espaços por marca de tabulação).

	$hello = "abc";
	echo $hello;

2. Todos os blocos de comando (ifs, while, etc) devem ser delimitados por
chaves. A chave que inicia o bloco deve estar na mesma linha do comando
de controle de fluxo. A chave que fecha o bloco deve estar em uma nova
linha, alinhada com o comando de controle de fluxo que iniciou o bloco.

	if ($hello == "abc") {
		echo $hello;
	}

3. O 'else' e 'elseif' devem iniciar após a chave que finaliza o bloco
anterior, na mesma linha.

	if ($hello == "abc") {
		echo $hello;
	} else {
		echo "Ni";
	}

4. Nos blocos de funções e classes, a chave que inicia o bloco deve ser
definida em uma nova linha, alinhada com a declaração da função ou classe.
A chave que encerra o bloco deve ser alinhada com a chave que inicia o
bloco.

	function helloworld()
	{
		echo "Hello world";
	}

5. Não utilizar espaço após o início ou antes de término de um parênteses.

	hello(a, b);

6. Utilizar espaço após cada vírgula que separe parâmetros de uma chamada
de função.

	hello(a, b);
