The error are sorted by their severity, identified by an non-negative
integer number. The lower the number, more dangerous is the issue. CoTeia
uses the following classification:

0:
- Invalid wikipage and swiki identification number: This is dangerous or
serious because the software should never issue an invalid number and, it
this happens, or it's a CoTeia's error or it's a malicious user sending
bogus identification numbers.

4:
- Incorrect password to access a wikipage or swiki.
